התפלגות אחידה

התפלגות אחידה היא התפלגות בה כל הערכים בטווח נתון בעלי הסתברות זהה להתרחש.

  • כל ערך בטווח בעל הסתברות זהה
  • שימושית למודלים של אקראיות מושלמת
  • בסיס לגנרטורי מספרים אקראיים

קוד Python:

import numpy as np import matplotlib.pyplot as plt # יצירת נתונים אחידים n_samples = 1000 uniform_data = np.random.uniform(0, 10, n_samples) # יצירת היסטוגרמה bins = np.linspace(0, 10, 21) hist, bin_edges = np.histogram(uniform_data, bins) print(f"ממוצע: {np.mean(uniform_data):.2f}") print(f"סטיית תקן: {np.std(uniform_data):.2f}")
