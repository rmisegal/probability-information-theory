אנטרופיה של שאנון

אנטרופיה מודדת את כמות האי-ודאות או המידע הממוצע בהתפלגות.

  • נוסחה: H(X) = -Σ p(x) log₂ p(x)
  • יחידות: ביטים (bits)
  • מקסימום: בהתפלגות אחידה
  • מינימום: כשיש ודאות מלאה

קוד Python:

import numpy as np # פונקציה לחישוב אנטרופיה def shannon_entropy(probabilities): # הוספת ערך קטן למניעת log(0) probs = np.array(probabilities) + 1e-10 entropy = -np.sum(probs * np.log2(probs)) return entropy # דוגמאות התפלגויות uniform = [0.25, 0.25, 0.25, 0.25] # אחידה skewed = [0.7, 0.2, 0.05, 0.05] # מוטה extreme = [0.97, 0.01, 0.01, 0.01] # קיצונית print(f"אחידה: {shannon_entropy(uniform):.2f}") print(f"מוטה: {shannon_entropy(skewed):.2f}") print(f"קיצונית: {shannon_entropy(extreme):.2f}")

אנטרופיה עבור התפלגויות שונות

ניתוח הגרף:
