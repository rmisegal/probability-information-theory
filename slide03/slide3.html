התפלגות נורמלית (גאוסיאנית)

התפלגות נורמלית היא ההתפלגות הנפוצה ביותר בטבע ובמדעי הנתונים.

  • צורת פעמון סימטרית
  • מוגדרת על ידי μ (ממוצע) ו-σ (סטיית תקן)
  • 68% מהנתונים בטווח μ±σ
  • 95% מהנתונים בטווח μ±2σ

קוד Python:

import numpy as np from scipy import stats # פרמטרים mu, sigma = 0, 1 # ממוצע וסטיית תקן x = np.linspace(-4, 4, 100) # חישוב פונקציית הצפיפות y = stats.norm.pdf(x, mu, sigma) # יצירת נתונים אקראיים samples = np.random.normal(mu, sigma, 1000) print(f"ממוצע: {np.mean(samples):.2f}")

עקומת פעמון אינטראקטיבית

ניתוח הגרף:

ציר X: ערכים | ציר Y: צפיפות הסתברות | אינטראקטיביות: שנה μ ו-σ לראות השפעה
