מדדי נטייה מרכזית

מדדי נטייה מרכזית מתארים את מרכז ההתפלגות של הנתונים.

  • ממוצע: סכום הערכים חלקי מספרם
  • חציון: הערך האמצעי כשהנתונים מסודרים
  • שכיח: הערך הנפוץ ביותר
  • רבעונים: חלוקה ל-4 חלקים שווים

קוד Python:

import numpy as np from scipy import stats # נתונים לדוגמה data = np.concatenate([ np.random.normal(5, 1, 300), np.random.normal(8, 0.5, 100), np.random.normal(12, 2, 50) ]) # חישוב מדדים mean_val = np.mean(data) median_val = np.median(data) mode_val = stats.mode(data)[0] q1 = np.percentile(data, 25) q3 = np.percentile(data, 75) print(f"ממוצע: {mean_val:.2f}") print(f"חציון: {median_val:.2f}")
