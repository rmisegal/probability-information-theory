שקף 2ב: היסטוגרמה של התפלגות אחידה

מושגי היסטוגרמה בהתפלגות אחידה:

  • היסטוגרמה: ייצוג גרפי של התפלגות הנתונים
  • Bins: מספר הקבוצות שמחלקות את הטווח
  • התפלגות אחידה: כל הערכים בטווח שווים בהסתברות
  • גובה עמודות: אמור להיות דומה בהתפלגות אחידה

קוד Python:

# יצירת היסטוגרמה של התפלגות אחידה import matplotlib.pyplot as plt import numpy as np # יצירת נתונים אחידים np.random.seed(42) uniform_data = np.random.uniform(0, 10, 1000) # יצירת היסטוגרמה counts, bins, patches = plt.hist(uniform_data, bins=20, alpha=0.7, edgecolor='black') # הוספת תוויות נתונים for i, count in enumerate(counts): plt.text(bins[i] + (bins[1]-bins[0])/2, count + 1, str(int(count)), ha='center', fontsize=8) plt.xlabel('Value') plt.ylabel('Frequency') plt.title('Histogram of Uniform Distribution')

היסטוגרמה של התפלגות אחידה (1000 דגימות)

ניתוח הגרף:

ציר X: ערכים (0-10) | ציר Y: תדירות
תוויות: מספר הדגימות בכל bin | מסקנה: גובה עמודות דומה = התפלגות אחידה
