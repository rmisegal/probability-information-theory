התפלגות פואסון

התפלגות פואסון מתארת מספר אירועים נדירים המתרחשים בפרק זמן או מרחב קבוע.

  • λ (למבדא) = קצב ממוצע של אירועים
  • מתאימה לאירועים נדירים ובלתי תלויים
  • דוגמאות: שיחות למוקד, תאונות דרכים
  • ממוצע = שונות = λ

קוד Python:

from scipy import stats import numpy as np # פרמטר λ (קצב אירועים) lambda_val = 3 # חישוב הסתברויות k_values = np.arange(0, 15) probabilities = stats.poisson.pmf(k_values, lambda_val) # יצירת נתונים אקראיים samples = np.random.poisson(lambda_val, 1000) print(f"ממוצע: {np.mean(samples):.2f}") print(f"שונות: {np.var(samples):.2f}")

התפלגות פואסון עם ערכי λ שונים

ניתוח הגרף:

ציר X: מספר אירועים (k) | ציר Y: הסתברות | מגמה: ככל ש-λ גדול יותר, ההתפלגות רחבה יותר
