שקף 1ב: תדירויות מסימולציה

חוק המספרים הגדולים קובע שככל שמספר הניסויים גדל, התדירות האמפירית מתקרבת להסתברות התיאורטית.

  • דגימות קטנות: סטיות גדולות מהתיאוריה
  • דגימות גדולות: התקרבות לערכים התיאורטיים
  • בסיס להסקה סטטיסטית

קוד Python:

# סימולציה של 1000 הטלות קובייה import numpy as np np.random.seed(42) rolls = np.random.randint(1, 7, 1000) # חישוב תדירויות unique, counts = np.unique(rolls, return_counts=True) frequencies = counts / 1000 # השוואה לתיאוריה theoretical = 1/6 for outcome, freq in zip(unique, frequencies): diff = freq - theoretical print(f"{outcome}: {freq:.3f} (diff: {diff:+.3f})")
