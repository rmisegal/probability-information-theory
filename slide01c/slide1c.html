שקף 1ג: היסטוגרמה של 1000 הטלות

היסטוגרמה היא ייצוג גרפי של התפלגות תדירויות הנתונים, המחולקים לקבוצות (bins).

  • גובה העמודה = תדירות התוצאה
  • רוחב העמודה = טווח הערכים
  • שטח כולל = מספר התצפיות הכולל

קוד Python:

# יצירת היסטוגרמה עם תוויות נתונים import matplotlib.pyplot as plt import numpy as np # נתוני הסימולציה rolls = np.random.randint(1, 7, 1000) unique, counts = np.unique(rolls, return_counts=True) # יצירת היסטוגרמה bars = plt.bar(unique, counts, alpha=0.7, edgecolor='black') # הוספת תוויות נתונים בראש כל עמודה for bar, count in zip(bars, counts): plt.text(bar.get_x() + bar.get_width()/2, bar.get_height() + 2, str(count), ha='center', fontweight='bold') plt.xlabel('Dice Outcome') plt.ylabel('Frequency') plt.title('Histogram of 1000 Dice Rolls')

ניתוח הגרף:

ציר X: תוצאות (1-6) | ציר Y: מספר הופעות
תוויות: מספר ההופעות מוצג בראש כל עמודה | מסקנה: התפלגות קרובה לאחידה
