שקף 1ג: היסטוגרמה של 1000 הטלות

היסטוגרמה היא ייצוג גרפי של התפלגות תדירויות הנתונים, המחולקים לקבוצות (bins).

  • גובה העמודה = תדירות התוצאה
  • רוחב העמודה = טווח הערכים
  • שטח כולל = מספר התצפיות הכולל

קוד Python:

# יצירת היסטוגרמה של 1000 הטלות import matplotlib.pyplot as plt import numpy as np # נתוני הסימולציה rolls = np.random.randint(1, 7, 1000) bins = np.arange(0.5, 7.5, 1) # 6 bins עבור 1-6 # יצירת היסטוגרמה plt.hist(rolls, bins=bins, alpha=0.7, edgecolor='black') plt.xlabel('Dice Outcome') plt.ylabel('Frequency') plt.title('Histogram of 1000 Dice Rolls') # הוספת קו תיאורטי expected = 1000/6 # ~166.67 plt.axhline(y=expected, color='red', linestyle='--')

הסתברות זריקת קובייה הוגנת

ניתוח הגרף:

ציר X: תוצאות (1-6) | ציר Y: הסתברות | מסקנה: כל תוצאה = 1/6 ≈ 0.167
