דיברגנס KL ואנטרופיה צולבת

דיברגנס KL מודד את המרחק בין שתי התפלגויות הסתברות.

  • KL(P||Q): מרחק מ-P ל-Q
  • אנטרופיה צולבת: H(P,Q) = H(P) + KL(P||Q)
  • שימוש: למידת מכונה ובחירת מודלים
  • ערך 0: התפלגויות זהות

קוד Python:

import numpy as np from scipy import stats # פונקציות חישוב def kl_divergence(p, q): return np.sum(p * np.log(p / q)) def cross_entropy(p, q): return -np.sum(p * np.log(q)) # התפלגות אמיתית p_true = stats.norm.pdf(x, 5, 1) p_true = p_true / np.sum(p_true) # קירובים שונים q1 = stats.norm.pdf(x, 5, 1.2) # קירוב טוב q2 = stats.norm.pdf(x, 4.5, 1.5) # קירוב בינוני print(f"KL1: {kl_divergence(p_true, q1):.3f}") print(f"KL2: {kl_divergence(p_true, q2):.3f}")
