שקף 2ה: פונקציית התפלגות מצטברת (CDF)

מה זה CDF?

  • CDF: Cumulative Distribution Function
  • הגדרה: F(x) = P(X ≤ x)
  • משמעות: הסתברות שהמשתנה ≤ ערך מסוים
  • טווח: 0 ≤ F(x) ≤ 1
  • מונוטונית עולה: אף פעם לא יורדת

נוסחת CDF להתפלגות אחידה:

F(x) = (x - a) / (b - a)
עבור a ≤ x ≤ b
עבור [0,10]: F(x) = x/10

תכונות CDF:

  • F(0) = 0: התחלה מאפס
  • F(10) = 1: סיום באחד
  • קו ישר: עלייה קבועה בהתפלגות אחידה
  • שיפוע: 1/(b-a) = 1/10 = 0.1

קוד Python:

# יצירת CDF להתפלגות אחידה import matplotlib.pyplot as plt import numpy as np from scipy import stats # יצירת נתונים אחידים np.random.seed(42) uniform_data = np.random.uniform(0, 10, 1000) # יצירת CDF תיאורטי x_theoretical = np.linspace(0, 10, 100) cdf_theoretical = x_theoretical / 10 # F(x) = x/10 # יצירת CDF אמפירי x_empirical = np.sort(uniform_data) y_empirical = np.arange(1, len(x_empirical) + 1) / len(x_empirical) # יצירת הגרף plt.figure(figsize=(10, 6)) plt.plot(x_theoretical, cdf_theoretical, 'r-', linewidth=2, label='Theoretical CDF') plt.plot(x_empirical, y_empirical, 'b-', alpha=0.7, label='Empirical CDF') plt.xlabel('x') plt.ylabel('F(x) = P(X ≤ x)') plt.title('CDF: Uniform Distribution [0,10]') plt.legend() plt.grid(True, alpha=0.3)

CDF: התפלגות אחידה [0,10]

ניתוח הגרף:

ציר X: ערכים (0-10) | ציר Y: הסתברות מצטברת (0-1)
קו אדום: CDF תיאורטי | קו כחול: CDF אמפירי | מסקנה: קו ישר = התפלגות אחידה
