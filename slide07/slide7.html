מדדי פיזור

מדדי פיזור מתארים עד כמה הנתונים מפוזרים סביב המרכז.

  • שונות: ממוצע ריבועי הסטיות מהממוצע
  • סטיית תקן: שורש השונות
  • שגיאת תקן: סטיית תקן חלקי √n
  • פיזור גדול = נתונים מפוזרים

קוד Python:

import numpy as np import matplotlib.pyplot as plt # יצירת 3 קבוצות נתונים עם פיזור שונה np.random.seed(42) data_a = np.random.normal(50, 5, 100) # פיזור קטן data_b = np.random.normal(50, 10, 100) # פיזור בינוני data_c = np.random.normal(50, 15, 100) # פיזור גדול # חישוב מדדי פיזור for name, data in [('A', data_a), ('B', data_b), ('C', data_c)]: std = np.std(data) var = np.var(data) print(f"{name}: σ={std:.1f}, σ²={var:.1f}")

השוואת פיזור בין קבוצות נתונים

ניתוח הגרף:

ציר X: ערכים | ציר Y: קבוצות | רוחב: מייצג את הפיזור - רחב יותר = פיזור גדול יותר
