התפלגות בינומית

התפלגות בינומית מתארת מספר הצלחות ב-n ניסויים בלתי תלויים, כאשר כל ניסוי בעל הסתברות p להצלחה.

  • n = מספר הניסויים
  • p = הסתברות להצלחה בניסוי יחיד
  • k = מספר ההצלחות (0 ≤ k ≤ n)
  • דוגמה: זריקת מטבע n פעמים

קוד Python:

from scipy import stats import numpy as np # פרמטרים n = 20 # מספר ניסויים p = 0.5 # הסתברות להצלחה # חישוב הסתברויות k_values = np.arange(0, n+1) probabilities = stats.binom.pmf(k_values, n, p) # ממוצע ושונות mean = n * p variance = n * p * (1 - p) print(f"ממוצע: {mean}, שונות: {variance}")

התפלגות בינומית עם ערכי p שונים

ניתוח הגרף:

ציר X: מספר הצלחות (k) | ציר Y: הסתברות | צבעים: כחול p=0.3, אדום p=0.5, ירוק p=0.7
